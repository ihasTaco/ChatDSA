ServerQuery is a Discord bot made with Discord.py that can query game servers using A2S (Valve Query API), FiveM, Minecraft, and SAMP. 
The bot provides server and player information in a customizable way, making it useful for Discord server owners who host dedicated servers. 
The bot also provides stability and customization options, and it can be fully customized for every server. 
The project is open-source and released under the MPL 2.0 license.
